# Cookie Viewer

**Your browser-based tool for understanding website cookies.**

Cookie Viewer is a **Chrome extension** that helps you:

- View cookies for any website instantly  
- Categorize cookies by source or expiration  
- Export cookies to CSV for analysis  
- Understand your web data while keeping your privacy protected  

---

### How to install

Cookie Viewer is available on the [Chrome Web Store](https://chrome.google.com/webstore/detail/cookie-viewer/pmonjppnoajddlblflnbodlaeaadldfc?hl=en).  
Click “Add to Chrome” to start using the extension in your browser.

---

### About

Perfect for developers, privacy enthusiasts, or anyone curious about their web data. All processing happens locally in your browser — no data leaves your device.
